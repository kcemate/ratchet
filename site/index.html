The loop

Three steps. Zero guesswork.

You don't write any code. Ratchet runs the loop. You review the PR.

1

Scan Free

Run npx ratchet scan in your project. Takes 30 seconds. You get a Production Readiness Score with a full breakdown — what's wrong, how bad, and in what category. No credit card. No account. Just answers.

2

Fix AI runs the loop

Run npx ratchet fix. Ratchet picks the highest-impact issues, writes the fixes, runs your tests to verify nothing broke, and commits only the changes that pass. If something fails, it rolls back automatically. Your code can only get better.

3

Ship Merge the PR

Ratchet opens a pull request with every fix, test result, and a before/after score. You read through it, approve what looks good, and merge. No mystery — you can see exactly what changed and why. The diff is yours.

Capabilities

More than a linter.

Ratchet is an AI-powered code quality engine that gets smarter with every run.

🐝
Swarm Mode

Launch multiple AI agents in parallel to fix different categories simultaneously. Fix testing, security, and types all at once — not one at a time.

🏗️
Architect Mode

An orchestrator agent plans the fix strategy, then delegates to specialized workers. Handles complex, cross-file refactors that single-pass tools miss.

📈
Incremental Scanning

Only re-scan files that changed since your last run. Fast feedback on every commit — seconds, not minutes. Works with git diff automatically.

🚦
Pre-commit Validation

Add Ratchet to your git hooks or CI pipeline. Block merges that drop your score below a threshold. Quality only goes up — never back down.

🔒
Score Regression Guard

Set a minimum score for your project. Ratchet prevents any change from lowering it. Like a ratchet wrench — it only turns one direction.

🧠
Cross-run Learning

Ratchet remembers what worked in previous runs. Fix patterns that succeeded get prioritized. Your project-specific context carries forward.

FAQ

Common questions

Do I need to know how to code?

Not really. If you can open a terminal and run a command, you can use Ratchet. You built your app with AI — we're just cleaning up what AI missed. You review the changes in a pull request, same as any other code review. If you don't understand something in the diff, that's okay — it still works.

What AI does it use?

Ratchet uses Claude (Anthropic) by default. On the Pro plan, AI is included — no API key needed. If you're on the free tier and want unlimited fixes, you can bring your own API key from Anthropic, OpenAI, or OpenRouter.

Is my code safe?

Yes. Your code never leaves your machine. The AI runs locally and makes calls to the AI provider directly from your computer — we never see your source code. We only communicate with our license server to count your fixes. That's it. No code uploaded, no data stored on our end.

What if something breaks?

It can't ship if it breaks. Ratchet runs your test suite after every fix — if tests fail, it rolls back that change automatically and moves on to the next one. Only fixes that pass your tests get committed. Your codebase can only get better, never worse. That's built into how it works, not just a promise.

How long does it take?

The free scan takes about 30 seconds. Each AI fix takes 1–3 minutes depending on your project size and test suite. A typical Pro session runs 20 fixes in about 30–45 minutes while you do something else. Come back to a pull request, review it, merge.

What languages and frameworks are supported?

Ratchet works with any JavaScript/TypeScript project out of the box — React, Next.js, Node, Express, you name it. Python support is coming soon. Since it uses AI to analyze and fix code, adding new languages is a matter of training, not rewriting rules. The score categories (testing, security, types, etc.) apply universally.
