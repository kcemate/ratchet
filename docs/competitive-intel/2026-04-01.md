- **CodeRabbit launched a proper CLI** (`cr review`) with agentic API keys and "unlimited reviews" pricing tier — directly targeting the same developer loop Ratchet lives in. ## 🟢 Gaps Ratchet Can Exploit

- **No one owns the iterative improvement loop.** CodeRabbit reviews. SonarQube flags. Claude Code fixes. But none of them do what Ratchet does: autonomous multi-iteration *improvement* cycles that commit working code and log progress. That loop is unclaimed territory.

- **CLI-native + offline/self-hosted story is weak across the board.** SonarQube is enterprise/hosted. CodeRabbit needs their service. Ratchet can own the "runs entirely local, no SaaS dependency" positioning.

- **MCP integration is a new distribution channel.** SonarQube is building an MCP server — Ratchet should too. Being queryable by AI agents (Claude Code, Cursor, etc.) is a new surface area none of the iterative improvement tools have yet.
