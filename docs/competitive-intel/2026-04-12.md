### Emerging Competitors to Watch:

- AI-powered technical audits for startups (code quality, security, scalability, architecture)
- Normally $15K, first customers at $10K
- One-week turnaround time

**vibescore** (open-source, highlighted April 11):
- Grades AI-generated ("vibe-coded") code on security, quality, dependencies, testing
- Supports Python, JS/TS, Rust, Go
- Pip-installable with zero dependencies

**Cline CLI 2.0** (gaining traction April 2026):
- Open-source, terminal-native AI coding agent with headless mode for CI/CD
- Free Minimax M2.5/Kimi K2.5 models, parallel agents
- Install: `npm install -g cline`, Review: `git diff | cline -y`

### Strategic Gaps for Ratchet to Exploit:

1. **Local-first/Ollama focus** - Most competitors are cloud-dependent; opportunity for privacy-conscious teams
2. **True CLI-native experience** - Many tools bolt CLI onto web platforms; Ratchet can lead with pure CLI workflow
3. **Agentic workflow integration** - Competitors focus on PR reviews; opportunity for pre-commit and local development hooks
4. **Cost transparency** - Enterprise pricing models unclear; opportunity for simple, predictable CLI tool pricing

### Trending Market Shifts:
- Multi-model reviews gaining traction (Copilot CLI pitting models against each other)
- Shift to local/offline-first tools to prevent code leakage
- Agentic workflows: Auto-fixes, PR monitoring, blast-radius analysis
