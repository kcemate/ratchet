# Competitive Intel — 2026-04-02

## Key Moves This Week

- **CodeRabbit shipped CLI v0.3.11 with `--agent` JSON mode (Mar 31)** — they're aggressively targeting the agent/CLI workflow. Also launched "Slop Detection" for AI-generated PRs (Mar 24) and audit logs for enterprise. Their Issue Planner → CodeRabbit Plan pipeline is maturing into a full SDLC tool, not just review. *Threat level: high. They're the closest CLI competitor and moving fast.*

- **DeepSource launched CLI v2 (Mar 7) + continuous CVE monitoring (Mar 28)** — CLI now queries findings, report cards, and vulns from terminal with JSON output. Their "AI Review" hybrid engine claims to beat CodeRabbit on benchmarks. Deprecated Free/Starter tiers — AI Review billed separately. *Gap: their CLI is query-focused, not iterative improvement like Ratchet.*

- **Qodo (formerly CodiumAI) raised $70M Series B** — positioning as AI code governance for enterprise (Walmart, NVIDIA, Ford). Claims to outperform Claude on code review benchmarks. Not CLI-first, but their funding means they'll expand fast. *Watch but don't panic — different market segment.*

- **PRGuard (open-source GitHub Action)** launched this week — free, MIT-licensed, 40+ rule checks to block AI slop PRs. Uses OpenAI/DeepSeek/Ollama. Zero-config. *Interesting adjacent tool, not a direct competitor but validates the "AI code quality" space is hot.*

- **Sourcery appears stagnant** — no notable 2026 releases found. Likely losing ground to CodeRabbit and DeepSource.

## Gaps Ratchet Can Exploit

1. **Iterative improvement loop** — CodeRabbit/DeepSource CLI tools find issues; Ratchet *fixes* them iteratively. Nobody else does the full analyze→fix→test→commit cycle from CLI.
2. **Local-first / offline** — competitors are all SaaS-dependent. Ratchet running locally with Ollama is a differentiator for privacy-conscious devs.
3. **Anti-slop positioning** — PRGuard and CodeRabbit's Slop Detection show demand. Ratchet could market as "fix slop before it ships" rather than just detecting it.
4. **No enterprise tax** — DeepSource killed free tier, CodeRabbit charges per seat. Ratchet's open CLI model is attractive for indie devs and small teams.

## Watch List

- **Qodo** — $70M war chest, enterprise focus, could move downmarket
- **CodeRabbit CLI** — closest competitor, shipping weekly
- **Readyish** — private beta scanning "vibe code" for production readiness (400k+ lines analyzed)
